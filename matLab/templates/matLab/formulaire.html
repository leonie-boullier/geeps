Linux

{% blocktrans %}Avec la version R2014a et supérieure, n'importe quelle adresse MAC peut servir comme HostID malgré le nom de l'interface. Si les interfaces sont énumérées, utiliser la plus petite interface.{% endblocktrans %}

{% blocktrans %}Pour obtenir l'adresse MAC grâce à la console shell, lancez la commande suivante :{% endblocktrans %}

/sbin/ifconfig {% blocktrans %}{% endblocktrans %}

{% blocktrans %}Où est le nom de l'interface réseau. Par exemple, en0, eth0, wlan0 ou enp5s0 sont des noms d'interface communs sur Linux, bien que cela peut varier d'un ordinateur à un autre.{% endblocktrans %}

{% blocktrans %}Pour la version R2013b et inférieure, l'hostID est l'adresse MAC du l'interface eth0 ou en0.{% endblocktrans %}

{% blocktrans %}Pour obtenir l'adresse MAC grâce à la console shell, lancez la commande suivante :{% endblocktrans %}

/{% blocktrans %}sbin/ifconfig eth0 ou /sbin/ifconfig eth1 {% endblocktrans %}

{% blocktrans %}L'adresse MAC est généralement précédée de la valeur 'HWaddr'. Si la distribution Linux n'a pas d'interface réseau qui suit le schéma (enX or ethX), il sera nécessaire de mettre à jour à la version R2014a ou supérieur, ou changez le nom des interfaces afin que MatLab le détecte. Consulter la documentation de la distribution Linux que vous possédez pour connaître les instructions afin de changer le nom de votre interface.{% endblocktrans %}

MacOS

{% blocktrans %}Pour MacOS, le hostID est l'adresse MAC du périphérique en0.{% endblocktrans %}

{% blocktrans %}Pour obtenir l'adresse MAC du périphérique en0, commencez par ouvrir un terminal :{% endblocktrans %}

{% blocktrans %}Ouvrir Finder{% endblocktrans %}

{% blocktrans %}Ouvrir le dossier Application{% endblocktrans %}

{% blocktrans %}Ouvrir le dossier Utilitaire{% endblocktrans %}

{% blocktrans %}Lancer Terminal{% endblocktrans %}

{% blocktrans %}Dans le terminal, rentrez la commande suivante:{% endblocktrans %}

ifconfig en0 | grep^ether

{% blocktrans %}L'adresse MAC est précédée par la valeur 'ether'.{% endblocktrans %}
